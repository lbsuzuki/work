UI

テクスチャ解像度切り替え

説明

  1. ヘルプ

    …melファイルやmelファイル保存フォルダを開きます。

  2. fileノード

    …解像度切り替えを適用するfileノードの範囲を設定できます。
    　●すべて　　…シーン内のすべてのfileノードを対象とします
    　●選択　　　…選択しているfileノードを対象とします

  3. 解像度状態

    …テクスチャの解像度を選択できます
    　●アニメーション　　…テクスチャを低解像度に切り替えます
    　●レンダリング　　　…テクスチャを高解像度に切り替えます

  4. 高解像度時のパネル状態

    …高解像度に切り替え時、すべてのモデルパネルを対象に表示切り替えを行います
    　●高解像度切り替え時 ワイヤーフレーム表示　　…モデルパネルをワイヤーフレーム表示にします(ショートカット【4】に相当)
    　●高解像度切り替え時 シェード表示　　　　　　…モデルパネルをシェード表示にします（ショートカット【5】に相当）
    　●切り替え時 何もしない

  5. 高解像度切り替え時 ハイパーシェードを閉じる

    …高解像度に切り替え時、ハイパーシェードを自動で閉じます

  6. 適用

    …設定をもとに高・低解像度切り替えを行います
