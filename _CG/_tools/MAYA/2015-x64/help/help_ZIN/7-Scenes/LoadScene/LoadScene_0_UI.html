UI

Bitmap Size

説明

  1. Image Plane List

    …イメージプレーンノード名をリスト化します。

  2. Reload

    …イメージプレーンリストを更新します。

  3. Width , Height

    …イメージプレーンに読み込んでいる画像の大きさを表示します。

  4. Output%

    …イメージプレーンに読み込んでいる画像の大きさを基準にレンダリングサイズをパーセンテージで指定します。

    　　　　Width　 ×　Output%　＝　レンダー設定のWidth
    　　　　Height　×　Output%　＝　レンダー設定のHeight

  5. Apply

    …レンダー設定のレンダリングサイズを指定されたサイズで編集します。

  6. OK

    …Applyと同じ作用とともにBitmapSizeウインドウを閉じます。

  7. Cancel

    …BitmapSizeウインドウを閉じます。
